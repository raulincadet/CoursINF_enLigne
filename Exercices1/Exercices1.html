Création d’objets

Les six exercices qui suivent portent sur la création de vecteurs, de matrices et de data frame.

  1. Créer un vecteur x, contenant ces valeurs: 1,2,4,3,5. Puis, afficher les données du vecteur 
x<-c(1,2,4,3,5)
print(x)
  1. Créer un vecteur y dont les valeurs vont de 35 à 100. Puis, afficher les données du vecteur. 
x<-35:100
print(x)    # pour afficher les données du vecteur x
  1. Créer un vecteur z qui contient que des nombre pairs allant de 24 à 80 
z<-seq(from=24,to=80,by=2)
print(z)          # pour afficher les données du vecteur z
  1. Créer un vecteur u dont les valeurs vont de 30 à 15 
u<-30:15
print(u)
  1. Créer un vecteur v dont les valeurs sont séquentielles, allant de 25 à 1, comme suit: 25,22,19 etc. 
v<-seq(from=25,to=1,by=-3)
print(v)
  1. Les vecteurs al et AL contiennent les 5 premières lettres de l’alphabet, respectivement en minuscule et en majuscule. Le vecteur note indique la plus petite note qui correspond à une lettre.
  1. Afficher les valeurs de ces deux vecteurs;
  2. Créer une matrice M avec les 3 vecteurs;
  3. Afficher les valeurs de la matrice M;
  4. Créer un data frame D avec les 3 vecteurs;
  5. Afficher les valeurs du data frame D.
al<-letters[1:5]   # creation d'un vecteur
AL<-LETTERS[1:5]
note<-seq(90,to=50,by=-10)
print(al)    # pour afficher les valeurs du vecteur al
print(AL)    # pour afficher les valeurs du vecteur AL
print(note)  # pour afficher les valeurs du vecteur note

M<-cbind(al,AL,note)
M           # afficher la matrice M
D<-data.frame(al,AL,note)
D           # afficher le data frame

Indexation en R

  1. Donner les noms qui suivent aux éléments du vecteur jo: Asefi, Mèsidye, Kòkòt, Debonè, Timafi. Puis, afficher les éléments du vecteur jo.
jo<-10:5 # creation d'un vecteur
names(jo)<-c('Asefi','Mèsidye','Kòkòt','Debonè','Timafi')
print(jo)
  1. La ligne de code ci-dessous crée une matrice du nom de Mat, avec les éléments d’un vecteur allant de 15 à 35.
  1. Afficher la matrice;
  2. Donner ces noms aux colonnes de la matrice, pour indiquer les variables: Score A, Score B, Score C;
  3. Donner ces noms créoles aux lignes de la matrice, pour indiquer les individus: Asefi, Mèsidye, Kòkòt, Debonè, Timafi, Dyesèl, Timafoun;
  4. Afficher de nouveau la matrice, pour vérifier les changements.
Mat<-matrix(15:35,ncol=3) # creation d'un vecteur
print(Mat)
colnames(Mat)<-c('Score A','Score B','Score C')
rownames(Mat)<-c('Asefi','Mèsidye','Kòkòt','Debonè','Timafi','Dyesèl','Timafoun')
print(Mat)
